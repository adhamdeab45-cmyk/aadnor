# قائمة الإطلاق النهائية

ضع علامة على كل بند قبل فتح الموقع للناس:

- [ ] نسخة احتياطية من Realtime Database وStorage.
- [ ] مشروع Firebase على Blaze وCloud Functions منشورة بنجاح.
- [ ] Google وPhone وEmail/Password مفعلة حسب الاستخدام.
- [ ] الدومين النهائي ضمن Authorized domains.
- [ ] أول أدمن مفعّل، وتم تسجيل الخروج والدخول من جديد.
- [ ] زر فحص الجاهزية في لوحة الأدمن لا يعرض أخطاء حرجة.
- [ ] طرق الدفع تحتوي حسابات وتعليمات حقيقية ثم تم تفعيلها يدوياً.
- [ ] تجربة إيداع وسحب وشحن وكيل بمبالغ صغيرة واختبارية.
- [ ] تجربة شراء تذكرة وجدولة نتيجة ونشرها مرة واحدة فقط.
- [ ] مراجعة الجوائز والأسعار ومواعيد تركيا لكل دورة.
- [ ] تجربة حساب مجمّد ووكيل موقوف وسحب صلاحية أدمن.
- [ ] مراجعة نصوص الشروط والخصوصية بواسطة مختص قانوني.
- [ ] إدخال بريد دعم ورقم/قنوات تواصل حقيقية.
- [ ] اختبار الهاتف والكمبيوتر واللغات الثلاث.
- [ ] تشغيل `npm run qa` قبل كل نشر.

**مهم:** التكامل الآلي مع شركة دفع محددة يحتاج حساب الشركة ومفاتيحها وWebhook الخاص بها. المشروع يجهز البنية الآمنة، لكنه لا يخترع بيانات مزود دفع غير موجودة.
