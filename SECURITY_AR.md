# ملاحظات أمان V302 Firebase Only

- لا توجد Cloud Functions في هذه النسخة.
- الصلاحيات تعتمد على Firebase Authentication وقواعد Realtime Database.
- حسابا الأدمن الأساسيان محددان بالبريد داخل القواعد والكود.
- إثباتات الإيداع لا يقبل إنشاؤها إلا داخل مجلد المستخدم وبحجم أقل من 8MB.
- الفائزون يظهرون بأسماء مخفية.
- جميع عمليات الأدمن والوكلاء تُسجل في مسارات التدقيق والسجلات.

## الحد المعروف

أي نظام مالي يعمل مباشرةً من المتصفح يمكن التلاعب بمنطقه بواسطة مستخدم متقدم حتى مع وجود قواعد جيدة. لذلك لا تعتبر هذه النسخة بديلاً نهائياً عن سيرفر للمال الحقيقي.
